# Anticipation-WebUI

Quick webui for Anticipation (https://github.com/jthickstun/anticipation/) more features will be added soon.

Open to PRs

# Installation

```
git clone https://github.com/jthickstun/anticipation/
pip install ./anticipation/
pip install -r ./anticipation/requirements.txt
git clone https://github.com/1aienthusiast/Anticipation-WebUI/
cd Anticipation-WebUI/
```
if you get an error like `ERROR: No matching distribution found for torch==2.0.1+cu118` then just do:
``` 
pip install torch 
```

# Models

Place models in the `models/` directory.
